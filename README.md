# Air_Quality_Analysis

📊 Project Overview

This project is an interactive Air Quality Index (AQI) dashboard built using HTML, CSS, and Chart.js to analyze pollution levels across major Indian cities in 2024.

It provides a clean, modern UI with dynamic charts and filters to explore air quality trends and environmental insights.

🌐 Live Features
Fully responsive web-based dashboard
Interactive filtering (City, Month, AQI Category)
Real-time chart updates using JavaScript
Smooth and minimal UI design
📁 Dataset Details
Time Period: January 2024 – May 2024
Cities Covered:
Bangalore
Chennai
Delhi
Hyderabad
Kolkata
Mumbai
Pune
Total Records: 150 readings
🌍 Parameters Included:
PM2.5
PM10
NO₂
SO₂
CO
O₃
Temperature
Humidity
Wind Speed
AQI Category
📌 Key Features
🔹 KPI Metrics
Average PM2.5 & PM10
Peak and Best Air Quality
Percentage of Unhealthy Days
Average NO₂ levels
🔹 Visualizations (Chart.js)
📊 Bar Chart → Avg PM2.5 by City
🍩 Donut Chart → AQI Category Distribution
📈 Line Chart → Monthly Trend by City
📊 Stacked Bar → AQI Count per City
📊 Grouped Bar → Pollutant Comparison (NO₂, SO₂, O₃)
⚫ Scatter Plot → Wind Speed vs PM2.5
🔹 Interactive Filters
City-wise filtering
Month-wise filtering
AQI category filtering
🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
Chart.js (for data visualization)
📈 Insights
A large portion of readings fall under Unhealthy AQI levels
Cities like Delhi and Pune show higher pollution peaks
Wind speed plays a role in pollutant dispersion
Pollution levels vary significantly across months
